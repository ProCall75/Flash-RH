# Glossaire métier — Flash Transports RH

> Termes métier utilisés dans le Portail RH

---

| Terme | Définition |
|-------|-----------|
| **RP** | Région Parisienne — zone géographique pour le calcul des frais (Repas midi RP = repas pris en Île-de-France) |
| **Province** | Toute zone hors RP — les frais en province ont des montants différents |
| **VL** | Véhicule Léger — catégorie de conducteur (utilitaires, fourgons) |
| **PL** | Poids Lourd — catégorie de conducteur (camions > 3,5T). Certains frais PL sont plus élevés (découché = 65€ vs 16€ VL) |
| **CP** | Congés Payés — absence rémunérée |
| **CP par anticipation** | Congés payés pris avant leur acquisition complète |
| **Congés sans solde** | Absence non rémunérée, à la demande du salarié |
| **AT** | Accident de Travail — absence déclarée suite à un accident professionnel |
| **Découché** | Nuit passée hors du domicile en déplacement professionnel — donne droit à une indemnité |
| **Casse-croûte** | Indemnité repas forfaitaire pour les conducteurs en déplacement (19€) |
| **Prime WE bloqué** | Indemnité versée quand le conducteur reste bloqué le week-end (120€) |
| **Cycle 20-20** | Période de frais allant du 20 du mois M au 20 du mois M+1 (pas du 1er au 31) |
| **J17** | Le 17ème jour de la période de frais — date d'alerte automatique si saisie incomplète |
| **Convention collective** | Accord sectoriel (transport routier) qui fixe les montants forfaitaires des frais et primes |
| **Contestation** | Action du conducteur qui refuse une correction apportée par l'admin à son relevé de frais |
| **Note de service** | Message officiel de la direction vers les employés (ex : rappel, info compta) |
| **TMS** | Transport Management System — logiciel de gestion du dispatching (Proaktou chez Flash) |
| **Cabinet de paie** | Prestataire externe qui gère les fiches de paie et le solde de congés |
| **RBAC** | Role-Based Access Control — système de droits par rôle (Admin, Bureau, Conducteur) |
| **PWA** | Progressive Web App — application web installable sur mobile sans passer par l'App Store |
| **RLS** | Row Level Security — sécurité au niveau des lignes de base de données (Supabase) |
